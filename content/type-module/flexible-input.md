```python
# Using Any for Flexible Input
```


```python
from typing import Any
```


```python
def stringify(value: Any) -> str:
    return str(value)
```


```python
stringify(42)
```




    '42'




```python
stringify([1, 2, 3])
```




    '[1, 2, 3]'
